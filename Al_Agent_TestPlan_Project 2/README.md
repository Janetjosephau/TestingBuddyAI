# AI Agent Test Planner Project 2

This project is a fresh implementation of an intelligent test planning agent using the B.L.A.S.T framework.

It supports:
- Jira and Rally runtime connections
- LLM-based test plan generation
- Preview-first output
- Template-driven test plan mapping

## Structure

- `architecture/` — project architecture, schemas, and SOPs
- `backend/` — Node/Express API and connector adapters
- `frontend/` — React + Vite UI
- `TestPlan_Template/` — source test plan template
- `UIScreenshots/` — design guidance for UI flow
- `Blast Framework/` — B.L.A.S.T. implementation guidance
